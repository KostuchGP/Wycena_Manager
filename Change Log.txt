Wycena Manager

File based saving of a resource structure in EXCEL.
This tool is used to manage calculate cost of design for single module from assembly of various devices.
---------------------------------------------------------

Changes

Version 1.1 Tool is released with stable state. You can use it on real device.

---------------------------------------------------------

Prerequisites

* Windows 7 Professional or upper
* Installed EXCEL 2016
* .NET Framework version 4.0 or upper
* .NET Microsoft.Office.Interop.Excel 15.0.0.0 or upper
---------------------------------------------------------

Installing - Not requested

---------------------------------------------------------

Running the tests

To check if your program working correctly, try to:

* kill all process with excel
*

If after this test there no warning system, just only Message Box with information, program working correctly.
